# Sales & Returns Analysis Dashboard

## Overview

This project is an interactive Power BI dashboard created to analyze sales performance, profitability, customer behavior, and product returns. It provides actionable business insights through dynamic visualizations, KPIs, and interactive filters to support data-driven decision-making.

## Features

**Overview Dashboard:**

Provides a high-level overview of key business metrics, including Total Sales, Total Profit, Profit Margin, and Total COGS, along with yearly sales and profit trends.

**Sales Analysis:**

Analyzes sales performance across regions, segments, categories, customers, and orders to identify top-performing products, customers, and business areas.

**Returns Analysis:**

Examines returned orders, returned sales, returned profit, and return rates, with detailed insights by region, ship mode, category, and year.

**Interactive Dashboard:**

Includes dynamic slicers for Region, Segment, Category, Year, and Month, enabling users to filter data and explore insights interactively.

**Business Insights:**

Delivers clear visualizations using KPI cards, bar charts, line charts, donut charts, treemaps, and tables to uncover trends and support strategic business decisions.
